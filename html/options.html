WordSave!

WordSave is a free pocket dictionary for people wanting to expand their vocabulary while also learning correct pronounciation.

It relies on a freely available API to work

dictionaryapi.dev

Used for word lookups - phonetic transcriptions and definitions.

How to Use

WordSave relies on keyboard shortcuts to make saving and viewing words easier:

  • Select a word and save using Control + Shift + S
  • View the WordSave dashboard with Control + Shift + L
